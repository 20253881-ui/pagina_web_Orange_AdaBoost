PAGINA WEB - ORANGE + ADABOOST + STREAMLIT

Proyecto:
Sistema Inteligente para la Predicción del Precio de Automóviles mediante Machine Learning

Archivos incluidos:
- app.py
- requirements.txt
- INSTRUCCIONES.txt

IMPORTANTE:
El archivo del modelo Orange no está incluido en este ZIP. Debes copiar tu modelo exportado desde Orange en esta misma carpeta y renombrarlo como:

modelo_orange.pkcls

También funciona si tu archivo se llama:
modelo_orange (3).pkcls
modelo_adaboost.pkcls

Cómo ejecutar:
1. Abre la carpeta del proyecto.
2. Copia dentro el archivo .pkcls exportado desde Orange.
3. Abre la terminal en esta carpeta.
4. Escribe:

pip install -r requirements.txt
streamlit run app.py

5. Se abrirá la página web en el navegador.
